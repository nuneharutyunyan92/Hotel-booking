Hotel Bookings SQL Analysis 

Overview

A SQL project based on the Kaggle hotel bookings dataset, analyzing booking
patterns, cancellations, and revenue-related questions using PostgreSQL
(analytical queries, views, functions, triggers).

Dataset


Source: [Kaggle link]
~119,000 rows, hotel booking records (2015–2017)
Key fields: hotel type, arrival date, lead time, cancellations, ADR, guests


What's included


schema.sql — table structure
queries/ — analytical queries (occupancy, cancellation rate, revenue by segment, etc.)
views.sql — reusable views
functions.sql — PL/pgSQL functions
triggers.sql — triggers


Key questions explored


What % of bookings get cancelled, by hotel type / lead time
Revenue trends by month / seasonality
Which guest segments bring the highest ADR
